Git practice

> git add [file name]
> git add .
> git commit
> git rm [file name]
> git status
> git log
# 一行で表示する
    > git log --oneline
#ファイルの差分を表示する
    > git log -p index.html
#行数を指定してログを表示
> git log -n 3
> git diff
#ステージとコミットの差分
    > git diff HEAD
> git remote add origin https://github.com/yamn15/intro_git
> git push origin master
